# 📊 AI Data Analyst

AI Data Analyst is a Streamlit-based intelligent data analysis web application that allows users to upload CSV datasets and analyze data using natural language questions.

The application automatically performs pandas operations, generates charts, and provides insights from the uploaded dataset.

---

# 🚀 Features

- Upload CSV files
- Natural language querying
- Smart sales/profit/customer analysis
- Automatic chart generation
- Interactive Streamlit UI
- Safe pandas execution
- Fast data visualization
- Deployable on Render

---

# 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
